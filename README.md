```
-----------------------------------------------------------------------------------------------
## For new Repository
----------------------------------------------------------------------------------------------
1.git init
2.echo "#test" >> README.md

5.git remote add origin https://github.com/MukulJaiswal/Test.git
6.git push -u origin master

-----------------------------------------------------------------------------------------------

----------------------------------------------------------------------------------------------
1. git add .
```
